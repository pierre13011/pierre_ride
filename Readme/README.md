
EcoRide

                                    Présentation


Objectifs du projet

- Proposer une interface simple et intuitive pour la recherche et la publication de trajets.  
- Favoriser le covoiturage écologique et économique.  



Technologies utilisées :

HTML CSS avec Bootstrap JavaScript 

Outils :
GitHub
Visual Studio Code
Navigateurs 

 Fonctionnalités prévues
    • Recherche de trajets par ville, date ou distance.
    • Mise en relation entre conducteurs et passagers.
    • Gestion des trajets à venir et passés.

Équipe

    • José : Directeur technique

    • Équipe EcoRide : Conception & développement web
    • Claude, Sophie,Bob, Alice : Conducteurs

Vision à long terme
EcoRide ambitionne de :
    • Devenir une référence française du covoiturage écologique.

    • Étendre la plateforme à l’échelle européenne.

    • Intégrer des fonctionnalités avancées (carte interactive, géolocalisation, paiements sécurisés).

    • Sensibiliser les utilisateurs à la mobilité durable.
